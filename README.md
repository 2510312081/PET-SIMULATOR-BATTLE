# Pet Simulator Battle

## Cara Instalasi
1. Pastikan Node.js sudah terinstall  
   Cek dengan:
   ```bash
   node -v
   npm -v
2. Clone repository: git clone https://github.com/2510312081/PET-SIMULATOR-BATTLE.git
3. Masuk ke folder project: cd PET-SIMULATOR-BATTLE
4. Install dependencies: npm install
5. Jalankan: npm start
6. Buka browser dan akses: http://localhost:3001
